# Koleno terapije

Kratek priročnik z vajami za koleno, pripravljen kot opomnik za izvajanje vaj po terapiji. Vsebina je razdeljena na dva dela:

- [raztezne vaje](prirocnik/raztezne_vaje.md) - 10 vaj za razteg zadnje in sprednje strani noge, narta, gležnja in kolena,
- [vaje za moč z elastiko](prirocnik/vaje_za_moc_z_elastiko.md) - 8 vaj za aktivacijo hoje, kolkov, stabilizatorjev kolena in funkcionalne moči.

Navodila so namenjena osebnemu opomniku in deljenju med osebami, ki izvajajo isti terapevtski program. Če se pojavi ostra bolečina, mravljinčenje, otekanje, nestabilnost kolena ali stopnjevanje težav po vadbi, vajo prekini in se posvetuj s fizioterapevtom ali zdravnikom.
